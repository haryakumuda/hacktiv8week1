=============

1. Create repository di Github
2. git config --global user.name "haryakumuda"
3. git config --global user.email "haryak279@gmail.com"
4. git config --global --list
5. git init                 (jika sudah pernah ada repository sebelumnya, maka tidak perlu lagi)           
6. git add *                (untuk menambahkan seluruh file)
   git add github.txt       (untuk menambahkan file github.txt saja)
7. git status
8. git commit -m "pesan yang akan dikirim"   
9. git remote add origin https://github.com/haryakumuda/hacktiv8week1.git
10. git push -u origin main   (nama branch)

untuk branching

1. git branch     (untuk cek daftar branch)
2. git branch "namabranch"  (untuk membuat branch baru)
3. git checkout "namabranch" (untuk pindah ke branch lain)
4. git add *
5. git commit -m "message"
6. git push -u origin namabranch
7. merge di github.com
8. git checkout main
9. git pull -u origin main
